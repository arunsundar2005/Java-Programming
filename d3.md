# Inheritance Programs
-----------------------------------------------------------------

## Program 1: Student → Exam → Result

Create a class **Student** with: - int roll - String name

Include methods to input and display student details.

Create a class **Exam** that inherits from `Student` with: - int marks1,
marks2, marks3

Include methods to input and display marks.

Create a class **Result** that inherits from `Exam` and: - Calculates
total and percentage - Displays complete result

### Sample Input:

Roll No: 101\
Name: Arun\
Marks1: 78\
Marks2: 85\
Marks3: 90

### Sample Output:

Roll No: 101\
Name: Arun\
Marks: 78 85 90\
Total: 253\
Percentage: 84.33%

------------------------------------------------------------------------

## Program 2: Employee → Salary

Create a class **Employee** with: - int empNo - String empName - double
basic

Create a class **Salary** that inherits from `Employee` and
calculates: - DA = 20% of basic - HRA = 15% of basic - Gross Salary =
basic + DA + HRA

### Sample Input:

Emp No: 201\
Name: Ravi\
Basic Salary: 25000

### Sample Output:

Employee No: 201\
Name: Ravi\
Basic: 25000\
DA: 5000\
HRA: 3750\
Gross Salary: 33750

------------------------------------------------------------------------

## Program 3: Publication → Book → Sales

Create a class **Publication** with: - String title - double price

Create a class **Book** that inherits from `Publication` with: - int
pages

Create a class **Sales** that inherits from `Book` with: - int qtySold -
Calculates total sale amount

### Sample Input:

Title: Java Programming\
Price: 450\
Pages: 350\
Quantity Sold: 10

### Sample Output:

Title: Java Programming\
Price: 450\
Pages: 350\
Quantity Sold: 10\
Total Sale: 4500

## Program 4: Account → SavingsAccount

Create a class **Account** with: - int accNo - String holderName -
double balance

Include methods to deposit and display balance.

Create a class **SavingsAccount** that inherits from `Account` and: -
Calculates interest at 5% per annum - Adds interest to balance -
Displays updated balance

### Sample Input:

Account No: 12345\
Name: Suresh\
Balance: 10000

### Sample Output:

Account No: 12345\
Name: Suresh\
Balance: 10000\
Interest: 500\
Updated Balance: 10500

## Program 5: Shape → Rectangle → Cuboid

Create a class **Shape** with: - double length - double breadth

Create a class **Rectangle** that inherits from `Shape` and: -
Calculates area

Create a class **Cuboid** that inherits from `Rectangle` with: - double
height - Calculates volume

### Sample Input:

Length: 5\
Breadth: 4\
Height: 3

### Sample Output:
