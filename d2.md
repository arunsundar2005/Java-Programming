# Day 2 -- Loops and Control Structure

## 1️⃣ Fibonacci Series

**Problem Statement:**\
Write a Java program to print the Fibonacci series up to **N terms**.\
The Fibonacci series is a sequence where each number is the sum of the
two preceding ones, starting from 0 and 1.

**Sample Input:** Enter the number of terms: 7

**Sample Output:** Fibonacci Series: 0 1 1 2 3 5 8

------------------------------------------------------------------------

## 2️⃣ Prime Numbers Till N

**Problem Statement:**\
Write a Java program to print all prime numbers from 1 to **N**.\
A prime number is a number greater than 1 that has no divisors other
than 1 and itself.

**Sample Input:** Enter the value of N: 20

**Sample Output:** Prime numbers till 20: 2 3 5 7 11 13 17 19

## 6️⃣ Palindrome Number

**Problem Statement:**\
Write a Java program to check whether a given number is a palindrome or
not.\
A palindrome number remains the same when reversed.

**Sample Input 1:** Enter a number: 121

**Sample Output 1:** 121 is a Palindrome number.

**Sample Input 2:** Enter a number: 123

**Sample Output 2:** 123 is NOT a Palindrome number.

------------------------------------------------------------------------

## 7️⃣ Armstrong Number

**Problem Statement:**\
Write a Java program to check whether a given number is an Armstrong
number.\
An Armstrong number is a number that is equal to the sum of its own
digits each raised to the power of the number of digits.

Example:\
153 → 1³ + 5³ + 3³ = 153

**Sample Input 1:** Enter a number: 153

**Sample Output 1:** 153 is an Armstrong number.

**Sample Input 2:** Enter a number: 123

**Sample Output 2:** 123 is NOT an Armstrong number.
