# Java Input / Output Practice Questions

## Question 2: Temperature Converter

Write a Java program that:

- Accepts temperature in **Celsius** from the user.
- Converts it into **Fahrenheit** using the formula:

\[
F = (C * 9/5) + 32
\]

- Display the converted temperature.

## Question 3: Simple Interest Calculator

Write a Java program that:

- Accepts the following inputs from the user:
  - Principal amount
  - Rate of interest
  - Time period
- Calculates the **Simple Interest** using:

\[
SI = \frac{P \times R \times T}{100}
\]

- Displays the calculated Simple Interest.

## Question 6: Electricity Bill Generator

Write a Java program that:

- Accepts the number of electricity units consumed.
- Calculates the electricity bill using the following conditions:

| Units Consumed | Cost per Unit |
|---------------|--------------|
| First 100 units | ₹2 per unit |
| Next 100 units | ₹3 per unit |
| Above 200 units | ₹5 per unit |

- Display the total electricity bill amount.

## Question 7: Student Grade System

Write a Java program that:

- Accepts marks of **five subjects** from the user.
- Calculates:
  - Total marks
  - Percentage
- Assign grades based on percentage:

| Percentage | Grade |
|------------|-------|
| 90 and above | A |
| 75 – 89 | B |
| 60 – 74 | C |
| 50 – 59 | D |
| Below 50 | Fail |

- Display total, percentage, and grade.
